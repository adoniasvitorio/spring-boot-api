# Desafio Técnico - Criação de Conta Bancária

## Visão Geral

Este desafio consiste em uma aplicação web parcialmente desenvolvida por uma IA utilizando React. Seu objetivo será utilizar a API previamente criada nesse desafio para se comunicar com essa app frontend, corrigir e melhorar o código existente, garantindo que os requisitos funcionais sejam atendidos com uma boa experiência ao usuário e que a base de código esteja pronta para futuras implementações.

## Requisitos Funcionais

### Criação de Conta

- Implementar o formulário de criação de conta, utilizando a API criada previamente.
- Exibir o saldo atual da conta após a criação

## Objetivos do Desafio

1. **Correção de Bugs**: Identificar possíveis erros presentes na implementação atual
2. **Refatoração**: Eliminar code smells e aplicar boas práticas de desenvolvimento
3. **Implementação**: Completar funcionalidades ausentes ou incorretas
